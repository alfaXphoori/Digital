## ส่วนที่ 3: เลขคณิตฐานสอง (20 นาที)

### แบบฝึก 3.1 — การบวกเลขฐาน 2

คำนวณและแสดงวิธีทำ (แสดง carry)

```
  1) 1010        2) 1101 0011
   + 0111         + 0010 1110
   ──────         ───────────
```

```
  3) 1111        4) 1010 1010
   + 0001         + 0101 0110
   ──────         ───────────
```

### แบบฝึก 3.2 — การลบเลขฐาน 2 (2's Complement)

จงหาค่า 2's Complement แล้วคำนวณ A − B โดยใช้วิธี A + (2's complement ของ B)

| ข้อ | A (ฐาน 2) | B (ฐาน 2) | 1's comp ของ B | 2's comp ของ B | A + 2's comp(B) | ผลลัพธ์ (ฐาน 10) |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| 1 | 1010 | 0011 | | | | |
| 2 | 1100 | 0101 | | | | |
| 3 | 1000 | 0110 | | | | |
